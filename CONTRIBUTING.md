# RULESET FOR RESPECTIVE CONTRIBUTIONS

### code improvement / additions

You can add and improve the existing code. 
- Make ensure it follows the existing structure.
- Adding comments is appreciated
- this also includes grammer correction and comments improvement

### Include dataset reference

- Continue the existing sequential structure.
- include necessary info like rows count, dimension of maze with reference to the data on internet.
- (optional) you can refer the author of the dataset too.
- don't forget to also explain the symbol references of the maze if changed.
- add the dataset reference [here](DATASETS.md)

### Your trained model

- you can include your notebook file in [models/](models/).
- create a separate folder inside [models/](models/) and name it as per your username so to prevent naming conflicts
- you can structure and add things inside your folder the way you feel better. Add separate README.md or anything necessary.
